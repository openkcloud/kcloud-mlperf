0. 클러스터와 연결된 서버에서 진행
1. pip install ansible로 환경 구축
2. kubespray/inventory/etri/hosts.yml 작성
    2-1. ansible_host에 각 노드의 IP 입력
    2-2. ansible_port에 각 노드의 ssh 접속 port 입력
    2-3. ansible_user에 각 노드의 ssh 접속 user 입력 (sudo 권한 있는 user)
    2-4. ansible_password와 ansible_become_password에 ssh 접속 password 입력
3. kubespray 디렉토리로 이동
4. ./99-conn-check.sh로 접속 테스트
5. ./01-provision.sh로 실행
